Local-only static demo

Chat against a local workflow.

This page is an optional tutorial UI served by the local static-files container. It uses only inline HTML, CSS, and JavaScript. No CDNs, fonts, analytics, or hidden remote imports are loaded.

No external assets No build step Editable endpoint

Edit the endpoint here or change DEFAULT_API_ENDPOINT near the top of this file. If your local API runs on a different port, it must allow browser requests from http://localhost:8080.

Request payload

The page sends JSON that is friendly to local n8n webhooks and simple local APIs. 

{
  "message": "hello",
  "chatInput": "hello",
  "history": [
    { "role": "user", "text": "..." },
    { "role": "assistant", "text": "..." }
  ],
  "sentAt": "2026-03-30T12:00:00.000Z",
  "source": "local-static-chat"
}

Local image or file URLs returned by the endpoint are rendered inline when they resolve to localhost, 127.0.0.1, the current page origin, or a relative local path. Remote URLs are left as plain text.
